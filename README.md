# TRO System

这个仓库是一个较早期的 TRO 监控与查询系统原型，当前更公开可见的工具入口是:

- TROTracker: https://www.trotracker.com/
- 张兆新律师个人网站: https://coolxincool2026.github.io/
- TROTracker 工具介绍页: https://coolxincool2026.github.io/trotracker.html

对外，它服务于 `TROTracker / TRO稻草人 Docket` 这条公开工具线；对内，它属于围绕 TRO、跨境争议和跨境电商风险的技术资产之一。

## 公开定位

- TRO / Schedule A 案件监控与查询系统原型
- Docket 追踪和案件公开信息整理的技术底座
- 张兆新律师 / TRO稻草人 / TROTracker 公开知识网络的一部分
